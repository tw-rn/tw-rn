Flex Shrink

React Native#

StyleProperties
flex-shrinkflexShrink: 1
flex-shrink-0flexShrink: 0

Usage#

<View style={tw`flex-shrink`}>
{/* ... */}
</View>

React Native Web#

StyleProperties
flex-shrinkflexShrink: 1
flex-shrink-0flexShrink: 0

Usage#

For web-only styles, you can use them with the web: platform pseudo selector.

<View style={tw`web:flex-shrink-0`}>
{/* ... */}
</View>

See in Tailwind website for more information here
