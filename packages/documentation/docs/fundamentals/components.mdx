For now, only the basic components are supported

## Basic Components

### `<View />`

```jsx
<View style={tw`items-center justify-center bg-gray-200`} />
```

### `<Text />`

```jsx
<Text style={tw`text-lg text-gray-500`} />
```

### `<Image />`

```jsx
<Image style={tw`h-12 w-12`} />
```

### `<TextInput />`

```jsx
<TextInput style={tw`text-gray-500 p-4`} />
```

### `<ScrollView />`

```jsx
<ScrollView style={tw`bg-gray-200 mx-4`} />
```
