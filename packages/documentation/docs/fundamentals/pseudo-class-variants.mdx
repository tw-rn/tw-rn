The reason that the Higher Order Component is to be able to implement pseudo-class variants

## Platform Variants

### Web

Add the `web:` prefix to apply a style when the platform is Web

```jsx
// bg-gray-200 will be applied to web
// others platforms will apply bg-gray-100 style
<View style={tw`bg-gray-100 web:bg-gray-200`} />

// bg-gray-200 will only be applied to web
// no styles will be applied to other platforms
<View style={tw`web:bg-gray-200`} />
```

### iOS

Add the `ios:` prefix to apply a style when the platform is iOS

```jsx
// bg-blue-200 will be applied to ios
// others platforms will apply bg-blue-100 style
<View style={tw`bg-blue-100 ios:bg-blue-200`} />

// bg-blue-200 will only be applied to ios
// no styles will be applied to other platforms
<View style={tw`ios:bg-blue-200`} />
```

### Android

Add the `android:` prefix to apply a style when the platform is Android

```jsx
// bg-green-200 will be applied to android
// others platforms will apply bg-green-100 style
<View style={tw`bg-green-100 android:bg-green-200`} />

// bg-green-200 will only be applied to android
// no styles will be applied to other platforms
<View style={tw`android:bg-green-200`} />
```

## Other Variants

### Focus

Add the `focus:` prefix to apply a style on focus

```jsx
// bg-teal-200 will be applied when the device is on focus mode
// once is on portrait mode, bg-teal-100 style will be applied
<View style={tw`bg-teal-100 focus:bg-teal-200`} />

// bg-teal-200 will only be applied to focus mode
// no styles will be applied when the device is on portrait mode
<View style={tw`focus:bg-teal-200`} />
```

### Hover

Add the `hover:` prefix to apply a style when an element is hover

**Note**: This only works in the web platform

```jsx
// bg-teal-200 will be applied when the device is on hover mode
// once is on portrait mode, bg-teal-100 style will be applied
<View style={tw`bg-teal-100 hover:bg-teal-200`} />

// bg-teal-200 will only be applied to hover mode
// no styles will be applied when the device is on portrait mode
<View style={tw`hover:bg-teal-200`} />
```
