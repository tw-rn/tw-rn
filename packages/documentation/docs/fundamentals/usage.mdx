title: Usage
---

**tw-rn** wraps basic React Native components with the Higher order Component `withTwrn`. For convenience, you can import the components already wrapped from the **tw-rn** module.

<!-- prettier-ignore-start -->
```jsx
import React from "react";
import { tw, View } from "tw-rn";

export default function Component() {
  return (
    <View style={tw`flex justify-center items-center`}>
      {/* ... */}
    </View>
  );
}
```
<!-- prettier-ignore-end -->
