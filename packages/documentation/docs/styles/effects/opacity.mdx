## React Native

| Style       | Properties     |
| :---------- | :------------- |
| opacity-100 | `opacity: 1`   |
| opacity-75  | `opacity: .75` |
| opacity-50  | `opacity: .5`  |
| opacity-25  | `opacity: .25` |
| opacity-0   | `opacity: 0`   |

### Usage

<!-- prettier-ignore-start -->
```jsx
<View style={tw`opacity-100`}>
  {/* ... */}
</View>
```
<!-- prettier-ignore-end -->

## React Native Web

| Style       | Properties     |
| :---------- | :------------- |
| opacity-100 | `opacity: 1`   |
| opacity-75  | `opacity: .75` |
| opacity-50  | `opacity: .5`  |
| opacity-25  | `opacity: .25` |
| opacity-0   | `opacity: 0`   |

### Usage

For web-only styles, you can use them with the `web:` platform pseudo selector.

<!-- prettier-ignore-start -->
```jsx
<View style={tw`web:opacity-75`}>
  {/* ... */}
</View>
```
<!-- prettier-ignore-end -->

See in Tailwind website for more information [here](https://tailwindcss.com/docs/opacity)
