## React Native

| Style       | Properties    |
| :---------- | :------------ |
| flex-grow   | `flexGrow: 1` |
| flex-grow-0 | `flexGrow: 0` |

### Usage

<!-- prettier-ignore-start -->
```jsx
<View style={tw`flex-grow`}>
  {/* ... */}
</View>
```
<!-- prettier-ignore-end -->

## React Native Web

| Style       | Properties    |
| :---------- | :------------ |
| flex-grow   | `flexGrow: 1` |
| flex-grow-0 | `flexGrow: 0` |

### Usage

For web-only styles, you can use them with the `web:` platform pseudo selector.

<!-- prettier-ignore-start -->
```jsx
<View style={tw`web:flex-grow-0`}>
  {/* ... */}
</View>
```
<!-- prettier-ignore-end -->

See in Tailwind website for more information [here](https://tailwindcss.com/docs/flex-grow)
