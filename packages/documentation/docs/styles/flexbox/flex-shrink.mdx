## React Native

| Style         | Properties      |
| :------------ | :-------------- |
| flex-shrink   | `flexShrink: 1` |
| flex-shrink-0 | `flexShrink: 0` |

### Usage

<!-- prettier-ignore-start -->
```jsx
<View style={tw`flex-shrink`}>
  {/* ... */}
</View>
```
<!-- prettier-ignore-end -->

## React Native Web

| Style         | Properties      |
| :------------ | :-------------- |
| flex-shrink   | `flexShrink: 1` |
| flex-shrink-0 | `flexShrink: 0` |

### Usage

For web-only styles, you can use them with the `web:` platform pseudo selector.

<!-- prettier-ignore-start -->
```jsx
<View style={tw`web:flex-shrink-0`}>
  {/* ... */}
</View>
```
<!-- prettier-ignore-end -->

See in Tailwind website for more information [here](https://tailwindcss.com/docs/flex-shrink)
