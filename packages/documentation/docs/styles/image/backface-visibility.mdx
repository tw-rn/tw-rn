### React Native

| Style            | Properties                      |
| :--------------- | :------------------------------ |
| backface-visible | `backfaceVisibility: "visible"` |
| backface-hidden  | `backfaceVisibility: "hidden"`  |

## Usage

<!-- prettier-ignore-start -->
```jsx
<Image style={tw`backface-visible`}/>
```
<!-- prettier-ignore-end -->

### React Native Web

Not supported.

Note: This works also for the `View` component.

See in React Native website for more information [here](https://reactnative.dev/docs/image-style-props#backfacevisibility)
