### React Native

| Style          | Properties              |
| :------------- | :---------------------- |
| resize-cover   | `resizeMode: "cover"`   |
| resize-contain | `resizeMode: "contain"` |
| resize-stretch | `resizeMode: "stretch"` |
| resize-repeat  | `resizeMode: "repeat"`  |
| resize-center  | `resizeMode: "center"`  |

## Usage

<!-- prettier-ignore-start -->
```jsx
<Image style={tw`resize-cover`}/>
```
<!-- prettier-ignore-end -->

### React Native Web

Not supported.

See in React Native website for more information [here](https://reactnative.dev/docs/image-style-props#resizemode)
