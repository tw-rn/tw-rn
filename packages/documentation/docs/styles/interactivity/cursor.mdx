### React Native

Not supported.

## React Native Web

| Style              | Properties              |
| :----------------- | :---------------------- |
| cursor-auto        | `cursor: 'auto'`        |
| cursor-default     | `cursor: 'default'`     |
| cursor-pointer     | `cursor: 'pointer'`     |
| cursor-wait        | `cursor: 'wait'`        |
| cursor-text        | `cursor: 'text'`        |
| cursor-move        | `cursor: 'move'`        |
| cursor-not-allowed | `cursor: 'not-allowed'` |

### Usage

For web-only styles, you can use them with the `web:` platform pseudo selector.

<!-- prettier-ignore-start -->
```jsx
<View style={tw`web:cursor-pointer`}>
  {/* ... */}
</View>
```
<!-- prettier-ignore-end -->

See in Tailwind website for more information [here](https://tailwindcss.com/docs/cursor)
