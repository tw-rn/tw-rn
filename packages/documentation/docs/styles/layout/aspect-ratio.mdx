### React Native

| Style   | Properties       |
| :------ | :--------------- |
| ratio-1 | `aspectRatio: 1` |

## Usage

<!-- prettier-ignore-start -->
```jsx
<Image style={tw`ratio-1 w-32`}>
  {/* ... */}
</Image>
```
<!-- prettier-ignore-end -->

### React Native Web

Not supported.

See in React Native website for more information [here](https://reactnative.dev/docs/layout-props#aspectratio)
