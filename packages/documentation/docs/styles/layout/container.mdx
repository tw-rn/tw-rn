### React Native

| Style     | Breakpoint | Properties      |
| :-------- | :--------- | :-------------- |
| container | None       | `width: '100%'` |

### React Native Web

| Style     | Breakpoint  | Properties           |
| :-------- | :---------- | :------------------- |
| container | None        | `width: '100%'`      |
|           | sm (640px)  | `maxWidth: '640px'`  |
|           | md (768px)  | `maxWidth: '768px'`  |
|           | lg (1024px) | `maxWidth: '1024px'` |
|           | xl (1280px) | `maxWidth: '1280x'`  |

## Usage

<!-- prettier-ignore-start -->
```jsx
<View style={tw`container`}>
  {/* ... */}
</View>
```
<!-- prettier-ignore-end -->

See in Tailwind website for more information [here](https://tailwindcss.com/docs/container)
