### React Native

| Style | Properties   |
| :---- | :----------- |
| z-0   | `zIndex: 0`  |
| z-10  | `zIndex: 10` |
| z-20  | `zIndex: 20` |
| z-30  | `zIndex: 30` |
| z-40  | `zIndex: 40` |
| z-50  | `zIndex: 50` |

## Usage

<!-- prettier-ignore-start -->
```jsx
<View style={tw`z-10`}>
  {/* ... */}
</View>
```
<!-- prettier-ignore-end -->

### React Native Web

| Style | Properties   |
| :---- | :----------- |
| z-0   | `zIndex: 0`  |
| z-10  | `zIndex: 10` |
| z-20  | `zIndex: 20` |

## Usage

For web-only styles, you can use them with the `web:` platform pseudo selector.

<!-- prettier-ignore-start -->
```jsx
<View style={tw`web:z-20`}>
  {/* ... */}
</View>
```
<!-- prettier-ignore-end -->

See in Tailwind website for more information [here](https://tailwindcss.com/docs/z-index)
