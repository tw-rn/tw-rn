### React Native

| Style      | Properties          |
| :--------- | :------------------ |
| max-h-full | `maxHeight: '100%'` |

### Usage

<!-- prettier-ignore-start -->
```jsx
<View style={tw`max-h-full`}>
  {/* ... */}
</View>
```
<!-- prettier-ignore-end -->

### React Native Web

| Style      | Properties          |
| :--------- | :------------------ |
| max-h-full | `maxHeight: '100%'` |

### Usage

For web-only styles, you can use them with the `web:` platform pseudo selector.

<!-- prettier-ignore-start -->
```jsx
<View style={tw`web:max-h-full`}>
  {/* ... */}
</View>
```
<!-- prettier-ignore-end -->

See in Tailwind website for more information [here](https://tailwindcss.com/docs/max-height)
