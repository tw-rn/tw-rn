### React Native

| Style      | Properties          |
| :--------- | :------------------ |
| min-h-0    | `minHeight: 0`      |
| min-h-full | `minHeight: '100%'` |

### Usage

<!-- prettier-ignore-start -->
```jsx
<View style={tw`min-h-0`}>
  {/* ... */}
</View>
```
<!-- prettier-ignore-end -->

### React Native Web

| Style      | Properties          |
| :--------- | :------------------ |
| min-h-0    | `minHeight: 0`      |
| min-h-full | `minHeight: '100%'` |

### Usage

For web-only styles, you can use them with the `web:` platform pseudo selector.

<!-- prettier-ignore-start -->
```jsx
<View style={tw`web:min-h-full`}>
  {/* ... */}
</View>
```
<!-- prettier-ignore-end -->

See in Tailwind website for more information [here](https://tailwindcss.com/docs/min-height)
