### React Native

| Style     | Properties    |
| :-------- | :------------ |
| text-xs   | `fontSize 12` |
| text-sm   | `fontSize 14` |
| text-base | `fontSize 16` |
| text-lg   | `fontSize 18` |
| text-xl   | `fontSize 20` |
| text-2xl  | `fontSize 24` |
| text-3xl  | `fontSize 30` |
| text-4xl  | `fontSize 36` |
| text-5xl  | `fontSize 48` |
| text-6xl  | `fontSize 64` |

### Usage

<!-- prettier-ignore-start -->
```jsx
<Text style={tw`text-xs`}>
  {/* ... */}
</Text>
```
<!-- prettier-ignore-end -->

### React Native Web

| Style     | Properties    |
| :-------- | :------------ |
| text-xs   | `fontSize 12` |
| text-sm   | `fontSize 14` |
| text-base | `fontSize 16` |
| text-lg   | `fontSize 18` |
| text-xl   | `fontSize 20` |
| text-2xl  | `fontSize 24` |

| text-4xl  | `fontSize 36` |
| text-5xl  | `fontSize 48` |
| text-6xl  | `fontSize 64` |

### Usage

For web-only styles, you can use them with the `web:` platform pseudo selector.

<!-- prettier-ignore-start -->
```jsx
<Text style={tw`web:text-sm`}>
  {/* ... */}
</Text>
```
<!-- prettier-ignore-end -->

See in Tailwind website for more information [here](https://tailwindcss.com/docs/font-size)
