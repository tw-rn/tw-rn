### React Native

| Style      | Properties           |
| :--------- | :------------------- |
| italic     | `fontStyle: italic'` |
| not-italic | `fontStyle: normal'` |

### Usage

<!-- prettier-ignore-start -->
```jsx
<Text style={tw`italic`}>
  {/* ... */}
</Text>
```
<!-- prettier-ignore-end -->

### React Native Web

| Style      | Properties           |
| :--------- | :------------------- |
| italic     | `fontStyle: italic'` |
| not-italic | `fontStyle: normal'` |

### Usage

For web-only styles, you can use them with the `web:` platform pseudo selector.

<!-- prettier-ignore-start -->
```jsx
<Text style={tw`web:non-italic`}>
  {/* ... */}
</Text>
```
<!-- prettier-ignore-end -->

See in Tailwind website for more information [here](https://tailwindcss.com/docs/font-size)
