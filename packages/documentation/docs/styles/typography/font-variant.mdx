### React Native

| Style                  | Properties                             |
| :--------------------- | :------------------------------------- |
| font-small-caps        | `fontVariant: [ "small-caps" ]`        |
| font-oldstyle-nums     | `fontVariant: [ "oldstyle-nums" ]`     |
| font-lining-nums       | `fontVariant: [ "small-caps" ]`        |
| font-tabular-nums      | `fontVariant: [ "tabular-nums" ]`      |
| font-proportional-nums | `fontVariant: [ "proportional-nums" ]` |

### Usage

<!-- prettier-ignore-start -->
```jsx
<Text style={tw`font-small-caps`}>
  {/* ... */}
</Text>
```
<!-- prettier-ignore-end -->

Not supported
