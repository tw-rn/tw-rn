### React Native

| Style          | Properties          |
| :------------- | :------------------ |
| font-hairline  | `fontWeight: '100'` |
| font-thin      | `fontWeight: '200'` |
| font-light     | `fontWeight: '300'` |
| font-normal    | `fontWeight: '400'` |
| font-medium    | `fontWeight: '500'` |
| font-semibold  | `fontWeight: '600'` |
| font-bold      | `fontWeight: '700'` |
| font-extrabold | `fontWeight: '800'` |
| font-black     | `fontWeight: '900'` |

### Usage

<!-- prettier-ignore-start -->
```jsx
<Text style={tw`font-normal`}>
  {/* ... */}
</Text>
```
<!-- prettier-ignore-end -->

### React Native Web

| Style          | Properties          |
| :------------- | :------------------ |
| font-hairline  | `fontWeight: '100'` |
| font-thin      | `fontWeight: '200'` |
| font-light     | `fontWeight: '300'` |
| font-normal    | `fontWeight: '400'` |
| font-medium    | `fontWeight: '500'` |
| font-semibold  | `fontWeight: '600'` |
| font-bold      | `fontWeight: '700'` |
| font-extrabold | `fontWeight: '800'` |
| font-black     | `fontWeight: '900'` |

### Usage

For web-only styles, you can use them with the `web:` platform pseudo selector.

<!-- prettier-ignore-start -->
```jsx
<Text style={tw`web:font-medium`}>
  {/* ... */}
</Text>
```
<!-- prettier-ignore-end -->

See in Tailwind website for more information [here](https://tailwindcss.com/docs/font-weight)
