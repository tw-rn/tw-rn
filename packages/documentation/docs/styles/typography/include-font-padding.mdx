### React Native

| Style             | Properties                  |
| :---------------- | :-------------------------- |
| font-padding      | `includeFontPadding: true`  |
| font-padding-none | `includeFontPadding: false` |

### Usage

<!-- prettier-ignore-start -->
```jsx
<Text style={tw`font-padding-none`}>
  {/* ... */}
</Text>
```
<!-- prettier-ignore-end -->

### React Native Web

Not supported

See in React Native website for more information [here](https://reactnative.dev/docs/text-style-props#includefontpadding)
