### React Native

| Style      | Properties       |
| :--------- | :--------------- |
| leading-3  | `lineHeight: 12` |
| leading-4  | `lineHeight: 16` |
| leading-5  | `lineHeight: 20` |
| leading-6  | `lineHeight: 24` |
| leading-7  | `lineHeight: 28` |
| leading-8  | `lineHeight: 32` |
| leading-9  | `lineHeight: 36` |
| leading-10 | `lineHeight: 40` |

### Usage

<!-- prettier-ignore-start -->
```jsx
<Text style={tw`leading-3`}>
  {/* ... */}
</Text>
```
<!-- prettier-ignore-end -->

### Usage

For web-only styles, you can use them with the `web:` platform pseudo selector.

<!-- prettier-ignore-start -->
```jsx
<Text style={tw`web:leading-4`}>
  {/* ... */}
</Text>
```
<!-- prettier-ignore-end -->

See in Tailwind website for more information [here](https://tailwindcss.com/docs/line-height)
