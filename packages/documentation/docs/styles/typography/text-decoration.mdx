### React Native

| Style               | Properties                                                                                                      |
| :------------------ | :-------------------------------------------------------------------------------------------------------------- |
| underline           | `textDecorationLine: "underline"` <br/> `textDecorationColor: "black"` <br/> `textDecorationStyle: "solid"`     |
| line-through        | `textDecorationLine: "line-through"` <br/> `textDecorationColor: "black"` <br/> `textDecorationStyle: "solid"`  |
| none                | `textDecorationLine: "none"` <br/> `textDecorationColor: "black"` <br/> `textDecorationStyle: "solid"`          |
| underline-double    | `textDecorationLine: "underline"` <br/> `textDecorationColor: "black"` <br/> `textDecorationStyle: "double"`    |
| line-through-double | `textDecorationLine: "line-through"` <br/> `textDecorationColor: "black"` <br/> `textDecorationStyle: "double"` |
| underline-dotted    | `textDecorationLine: "underline"` <br/> `textDecorationColor: "black"` <br/> `textDecorationStyle: "dotted"`    |
| line-through-dotted | `textDecorationLine: "line-through"` <br/> `textDecorationColor: "black"` <br/> `textDecorationStyle: "dotted"` |
| underline-dashed    | `textDecorationLine: "underline"` <br/> `textDecorationColor: "black"` <br/> `textDecorationStyle: "dashed"`    |
| line-through-dashed | `textDecorationLine: "line-through"` <br/> `textDecorationColor: "black"` <br/> `textDecorationStyle: "dashed"` |

### Usage

<!-- prettier-ignore-start -->
```jsx
<Text style={tw`underline`}>
  {/* ... */}
</Text>
```
<!-- prettier-ignore-end -->

### React Native Web

| Style        | Properties                                                                                                     |
| :----------- | :------------------------------------------------------------------------------------------------------------- |
| underline    | `textDecorationLine: "underline"` <br/> `textDecorationColor: "black"` <br/> `textDecorationStyle: "solid"`    |
| line-through | `textDecorationLine: "line-through"` <br/> `textDecorationColor: "black"` <br/> `textDecorationStyle: "solid"` |
| none         | `textDecorationLine: "none"` <br/> `textDecorationColor: "black"` <br/> `textDecorationStyle: "solid"`         |

### Usage

For web-only styles, you can use them with the `web:` platform pseudo selector.

<!-- prettier-ignore-start -->
```jsx
<Text style={tw`web:line-through`}>
  {/* ... */}
</Text>
```
<!-- prettier-ignore-end -->

See in Tailwind website for more information [here](https://tailwindcss.com/docs/text-decoration)
