### React Native

| Style             | Properties                                   |
| :---------------- | :------------------------------------------- |
| text-shadow-sm    | `textShadowOffset: { width: 0, height: 1 }`  |
| text-shadow       | `textShadowOffset: { width: 0, height: 2 }`  |
| text-shadow-md    | `textShadowOffset: { width: 0, height: 4 }`  |
| text-shadow-lg    | `textShadowOffset: { width: 0, height: 10 }` |
| text-shadow-xl    | `textShadowOffset: { width: 0, height: 20 }` |
| text-shadow-2xl   | `textShadowOffset: { width: 0, height: 25 }` |
| text-shadow-nonde | `textShadowOffset: { width: 0, height: 0 }`  |

### Usage

<!-- prettier-ignore-start -->
```jsx
<Text style={tw`text-shadow-sm`}>
  {/* ... */}
</Text>
```
<!-- prettier-ignore-end -->

### React Native Web

Not supported

See in React Native website for more information [here](https://reactnative.dev/docs/text-style-props#textshadowoffset)
