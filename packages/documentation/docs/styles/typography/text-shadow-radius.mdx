### React Native

| Style                   | Properties             |
| :---------------------- | :--------------------- |
| text-shadow-radius-sm   | `textShadowRadius: 1`  |
| text-shadow-radius      | `textShadowRadius: 2`  |
| text-shadow-radius-md   | `textShadowRadius: 4`  |
| text-shadow-radius-lg   | `textShadowRadius: 10` |
| text-shadow-radius-xl   | `textShadowRadius: 20` |
| text-shadow-radius-2xl  | `textShadowRadius: 25` |
| text-shadow-radius-none | `textShadowRadius: 0`  |

### Usage

<!-- prettier-ignore-start -->
```jsx
<Text style={tw`text-shadow-radius-sm`}>
  {/* ... */}
</Text>
```
<!-- prettier-ignore-end -->

### React Native Web

Not supported.
