### React Native

| Style               | Properties                 |
| :------------------ | :------------------------- |
| text-ltr            | `writingDirection: "ltr"`  |
| text-rtl            | `writingDirection: "rtl"`  |
| text-direction-auto | `writingDirection: "auto"` |

### Usage

<!-- prettier-ignore-start -->
```jsx
<Text style={tw`text-lrt`}>
  {/* ... */}
</Text>
```
<!-- prettier-ignore-end -->

### React Native Web

Not supported.

See in React Native website for more information [here](https://reactnative.dev/docs/text-style-props#writingdirection)
